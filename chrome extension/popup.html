goshxt

Navigate to the NDHU course registration page to use this extension.

Base URL:
https://sys.ndhu.edu.tw/

How to use:
1. Switch to the pre-select table
2. Click "Fetch Courses" to scan available courses
3. Click "Execute" to register for all courses

This script scans the lower table so make sure you're at the correct place when you scan

Some operation results can be seen in console, including course adding results.
