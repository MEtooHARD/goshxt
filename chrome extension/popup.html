goshxt

Go to the course registration page and login.
A panel should automatically show up.

You can also force active if it doesn't show up.
This only works under https://sys.ndhu.edu.tw/
How to use:
1. Switch to the pre-select table
2. Click "Fetch Courses" to scan available courses
3. Click "Execute" to register for all courses

Manual Inject:
Use the button above if the helper doesn't appear automatically.

This tool scans any lower table so make sure you're at the correct place when you do fetch.

Some operation results can be seen in console, including course adding results.
